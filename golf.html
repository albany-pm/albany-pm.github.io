Coming Soon: PerlGolf as a tool to learning Perl
This month's contest is to generate Box Fractals.
Please follow the generic Perl Golf rules
Your script must generate the appropriate box fractal given a single argument. For example, given 0, your program should generate: 
#
given 1: 
# #
 # 
# #
given 2: 
# #   # #
 #     # 
# #   # #
   # #   
    #    
   # #   
# #   # #
 #     # 
# #   # #
Note that until recently, trailing spaces were incorrectly omitted in the sample output above. They are required, and have been added into the samples.
You can use the following test script to verify your program functions properly before submission. The script explicitly tests the input values 0 to 6 (previously 8, reduced to increase judge sanity). What you generate on extraneous input doesn't matter. To run the test script, uncompress it, and run it, passing in the name of your script as the sole argument.
Entries with the same number of strokes will be further ranked based on the number of alphanumeric characters - the fewer the better.
Please email all solutions to will-perlgolf at coleda.com - I'll keep track manually and post results (but not code) until the contest closes at 7pm ET on December 10th. You don't have to be a member of albany PM to play.
See the Leaderboard for scores.
